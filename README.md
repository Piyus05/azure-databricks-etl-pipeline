
# Azure Databricks ETL Pipeline (Simulated)

This project demonstrates an end-to-end ETL pipeline using Spark concepts:
Raw CSV → Bronze → Silver → Gold.

## Tech Stack
- PySpark
- SQL
- Delta Lake (simulated)
- Azure Architecture (conceptual)

## Pipeline Flow
Source CSV → Ingestion → Cleaning → Transformation → Aggregation

## How to Run
Use any PySpark environment or Databricks notebook.
